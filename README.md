## В чем была задача?
Нужно было разработать решение для постоянного отслеживания метрик соединения (задержка, джиттер...) между двумя сетевыми устройствами. Для соединения устройств использовались Smart SFP модули нашего производства. Реализовать интеграцию с Zabbix.
## Принцип работы
Для отслеживания метрик используется протокол TWAMP (описан в [RFC 5357](https://datatracker.ietf.org/doc/html/rfc5357)). Скрипт, который работает на модулях, основан на [twampy](https://github.com/nokia/twampy/tree/master).
В измерениях участвуют оба модуля, но один из них, помимо этого, также осуществляет отправку данных на Zabbix-сервер. Для этого используются [zabbix_utils](https://github.com/zabbix/python-zabbix-utils/tree/main?tab=readme-ov-file) . На Zabbix-сервер предварительно нужно загрузить template.
